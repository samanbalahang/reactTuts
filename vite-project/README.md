# دوره آموزش React

## فهرست مطالب

### فصل اول: مبانی
- مقدمه‌ای بر React و اکوسیستم آن
- آشنایی با مفاهیم پایه‌ای توسعه وب مدرن

### فصل دوم: پایه‌های React و JSX
- درک مفهوم کامپوننت‌نویسی
- آشنایی با سینتکس JSX و قوانین آن
- کار با رویدادها و رندر شرطی

### فصل سوم: مدیریت State و Props
- آشنایی با Props و ارسال داده بین کامپوننت‌ها
- مدیریت State با هوک useState
- تفاوت‌های State و Props

### فصل چهارم: هوک‌های حیاتی و اثرات جانبی
- مدیریت اثرات جانبی با useEffect
- کار با هوک‌های useContext و useRef
- آشنایی با useReducer

### فصل پنجم: مدیریت State سراسری
- کار با Context API
- مدیریت State در سطح برنامه
- ترکیب Context با useReducer

### فصل ششم: مسیریابی با React Router
- راه‌اندازی Router در برنامه
- ایجاد مسیرهای داینامیک
- مدیریت Navigation و پارامترها

### فصل هفتم: کار با فرم‌ها و مدیریت آن‌ها
- ایجاد فرم‌های کنترل شده
- اعتبارسنجی فرم‌ها
- مدیریت وضعیت ارسال فرم

### فصل هشتم: بهینه‌سازی و ساخت
- بهینه‌سازی Performance
- ساخت نسخه Production
- آنالیز باندل خروجی

### فصل نهم: تست‌نویسی
- آشنایی با اصول تست‌نویسی
- نوشتن تست برای کامپوننت‌ها
- تست تعاملات کاربر

### فصل دهم: پروژه نهایی و جمع‌بندی
- پیاده‌سازی پروژه کاربردی
- جمع‌بندی کلی مباحث دوره
- معرفی مسیرهای ادامه یادگیری
